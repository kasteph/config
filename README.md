# config

Clone into `~/.config`: `git clone git@github.com:stephsamson/config.git ~/.config`.

* emacs

## Symlinks

From `$HOME`:

``` sh
$ ln -Ts ~/.config/.$.doom/ .
$ ln -s ~/.config/.gitconfig .
$ ln -s ~/.config/.bash* .
$ ln -s ~/.config/.tmux.conf .
```
